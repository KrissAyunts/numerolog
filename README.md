# Нумерологический Калькулятор

Современный веб-сайт для расчета психоматрицы Пифагора по дате рождения.

## Как запустить

### Вариант 1 (Простой)
Просто откройте файл `index.html` в вашем браузере.

### Вариант 2 (Для разработки)
Если вы хотите использовать локальный сервер (рекомендуется для корректной работы шрифтов и обновлений):

1. Установите зависимости (если есть `package.json`):
```bash
npm install
```

2. Запустите сервер:
```bash
npx vite
```

## Структура проекта
- `index.html` - Основной файл разметки
- `style.css` - Стили (Dark Theme, Glassmorphism)
- `script.js` - Логика расчета матрицы Пифагора
